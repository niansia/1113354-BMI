# BMI 計算機（WinForms / .NET Framework 4.7.2）

這個專案是我用 `C# WinForms` 做的 BMI 計算機，目標是把「功能完整」和「介面美觀」一起做好。

## 我做了哪些功能

- 我可以輸入身高（cm）和體重（kg）來計算 BMI。
- 我有加上防呆驗證，像是：
  - 不能輸入空值
  - 不能輸入非數字
  - 不能輸入 0 或負數
- 我會顯示 BMI 到小數點後 2 位。
- 我會顯示 BMI 判定：過輕、正常、過重、輕度肥胖、中度肥胖、重度肥胖。
- 我會依照身高算出正常體重範圍。
- 我有做 BMI 視覺化進度顯示與分佈位置提示。
- 我有做風險等級與建議文字。
- 我有做歷史紀錄清單（最近幾筆計算）。
- 我有做「複製本次報告」功能。
- 我有做深色 / 淺色模式切換。

## 操作方式

- 在「身高(cm)」輸入例如 `170`
- 在「體重(kg)」輸入例如 `65`
- 按「計算」就會看到完整分析
- 按「清除」可以重填

## 鍵盤快速操作

- `Tab`：切換欄位
- `Enter`：直接計算
- `Esc`：快速清除

## BMI 判定標準

- `< 18.5`：過輕
- `18.5 ~ 23.9`：正常
- `24 ~ 26.9`：過重
- `27 ~ 29.9`：輕度肥胖
- `30 ~ 34.9`：中度肥胖
- `>= 35`：重度肥胖

## 計算公式

`BMI = 體重(kg) / [身高(m) × 身高(m)]`

我在程式裡會先把公分轉成公尺再計算。
