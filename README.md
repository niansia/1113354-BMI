# BMI 計算機（WinForms / .NET Framework 4.7.2）

這是一個使用 `C# WinForms` 開發的 BMI 計算機，提供：

- 身高、體重輸入
- 防呆輸入驗證（非數字、0、負數，欄位旁即時提示）
- BMI 計算（顯示到小數點後 2 位）
- BMI 體位判定（過輕、正常、過重、輕度肥胖、中度肥胖、重度肥胖）
- 正常體重範圍建議（依輸入身高自動計算）
- 視覺化 BMI 進度條顯示
- 清除功能與操作快捷鍵（`Enter` 計算、`Esc` 清除）
- 輸入限制（只接受數字與一個小數點）
- ToolTip 使用提示

## 執行方式

1. 使用 Visual Studio 開啟專案（`.csproj`）。
2. 以 `Debug` 或 `Release` 建置。
3. 按 `F5` 執行。

---

## 使用說明

1. 在「身高(cm)」輸入身高（例如：170）。
2. 在「體重(kg)」輸入體重（例如：65）。
3. 按下「計算」按鈕（或直接按 `Enter`）。
4. 查看 BMI 數值與判定結果。
5. 需要重算時可按「清除」按鈕（或按 `Esc`）。

---

## BMI 判定標準

- `< 18.5`：過輕
- `18.5 ~ 23.9`：正常
- `24 ~ 26.9`：過重
- `27 ~ 29.9`：輕度肥胖
- `30 ~ 34.9`：中度肥胖
- `>= 35`：重度肥胖

---

## BMI 計算公式

`BMI = 體重(kg) / [身高(m) × 身高(m)]`

> 程式中會先將公分轉換成公尺再計算。

---

## UI 設計重點

- 使用一致字型（微軟正黑體）
- 主要按鈕高對比配色（計算）
- 結果區塊分成「BMI 數值」與「體位判定」
- 使用顏色提示判定狀態（正常綠色、警示橘紅色等）
- 結果區塊增加 BMI 進度條與正常體重範圍
- 底部提示可快速引導鍵盤操作

---

## 鍵盤操作與可用性

- `Tab`：依序切換輸入欄位與按鈕
- `Enter`：直接計算
- `Esc`：快速清除
- `ErrorProvider`：錯誤直接顯示在欄位旁，減少彈窗干擾
